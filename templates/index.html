Phishing Website Detector

Upload your website feature file (.csv) and get the prediction result in an Excel file.

How to use

  1. Prepare your website feature file in CSV format.
  2. Make sure the CSV contains the same feature columns used in training.
  3. Click Choose File and upload your CSV.
  4. Click Predict.
  5. The system will download an Excel output with predictions.

Accepted file type: .csv | Example: new_website_features.csv
